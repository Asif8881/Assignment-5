- Question-3: What is event delegation in the DOM?

-Ans:Event delegation is a technique where a single event listener is added to a parent element instead of multiple child elements. This works because of event bubbling, where events triggered on child elements propagate up to their ancestors.

- Question-4: What is event bubbling in the DOM?

-Ans:vent bubbling is a behavior where an event starts at the target element (where the event occurred) and then propagates upwards through its ancestor elements (parent, grandparent, etc.).
